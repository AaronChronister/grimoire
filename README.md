# Grimoire
Grimoire is a standalone deck building fantasy card game inspired by standard TCG draft formats (e.g. Magic the Gathering Limited). Players draft cards one at a time from randomized packs, passing the packs around the table until all cards have been chosen. 

# Drafting Grimoire with Draftmancer
Draftmancer is a free browser based tool for running TCG drafts with friends online. It is primarily focused on MTG limited formats, but also supports custom card sets. To draft Grimoire using Draftmancer, navigate to (https://draftmancer.com/) in your broswer and select the `Settings` tab, then upload the `grimoire_draftmancer.txt` file from this repo into the Custom Card List box at the bottom of the page: <img width="3310" height="1698" alt="Draftmancer" src="https://github.com/user-attachments/assets/89f5c962-2235-48c9-a7be-cf21ef99af14" />
That's it! Just invite friends to the draftmancer lobby and start a Grimoire draft.
